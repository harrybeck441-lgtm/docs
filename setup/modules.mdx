## 🤔 What's Inside?

Following are the modules and their features.

### Contact Management

- Add contacts individually or import contacts in bulk (CSV/XLSX support)
- Tagging and segmentation of contacts for targeted campaigns

### Campaign Creation

- Use pre-designed templates or import custom templates
- Schedule campaigns or send them immediately

### Email Delivery

- Send emails through AWS SES, leveraging its secure and reliable infrastructure

### Campaign Analytics

- Real-time tracking of emails with detailed stats:
  - Open rate
  - Click rate
  - Bounce rate
- Integration with AWS SES for email delivery and tracking

### Additional Features

- **Express Server**: Handles API requests and serves static files from the `dist` directory
- **React SPA**: Provides a responsive and dynamic user interface
- **Environment Configuration**: Easy-to-set-up environment variables for sensitive data
- **Social Login**: Authentication with Google, Microsoft, and Apple accounts
