## 🎬 Loom 3 — AI Voice Agents: Inbound & Outbound Calling

[Watch on Loom](YOUR_LOOM_LINK_HERE)

### What's Covered

This video explains how to build, configure, and deploy AI voice agents for both inbound and outbound calling. It covers agent setup, voice selection, call behavior settings, and how to review call performance in the logs.

### Key Features

**Agent Library**

- Browse all agents in a card or table layout
- Search by name or filter by type, status, or tags
- View performance at a glance: total calls, successful calls, failed calls, and last active time

**Creating an AI Agent**

- Choose an LLM model and tune the response creativity (temperature)
- Select a voice from 12\+ pre-built options across providers (Cartesia, ElevenLabs, Deepgram)
- Configure speech-to-text (STT) provider for live transcription
- Write or auto-generate a system prompt defining agent personality and goals

**Call Behavior Settings**

- Set maximum call duration (default 5 minutes)
- Enable voicemail detection to skip unanswered calls
- Adjust silence threshold to control conversational pacing
- Add background ambient sounds for a natural call environment
- Enable call recording and live transcription

**Welcome Message Options**

- Silent start: wait for the contact to speak first
- Dynamic: AI generates an opening based on contact context
- Defined: use a fixed, scripted opening message

**Inbound Call Routing**

- Assign a specific AI agent to handle calls on any phone number
- Each number can have a dedicated inbound agent and outbound agent

**Outbound Calling**

- Launch outbound calls manually from a contact record or via workflow automation
- Pass live contact data to the agent dynamically during the call
- Credit checking and call cost tracking before each call

**Call Logs & Analytics**

- Full call history: date, duration, cost, call type (inbound/outbound)
- Call outcome: success or failure with disconnection reason
- Sentiment analysis on each call
- Improvement suggestions with priority levels (High, Medium, Low)

## 🎬 Loom 4 — Automations & Workflows

[Watch on Loom](YOUR_LOOM_LINK_HERE)

### What's Covered

This video shows how to build automated workflows using the visual drag-and-drop builder. It covers every node type, how to set triggers, schedule workflows, and monitor live executions — all without writing a single line of code.

### Key Features

**Visual Workflow Builder**

- Drag-and-drop canvas to design multi-step automation flows
- Connect nodes visually with lines to define the execution path
- Zoom, pan, and reorganize your workflow freely

**Node Types (Building Blocks)**

- **Trigger** — the starting point that kicks off the workflow
- **AI Agent** — place an outbound call or run an AI interaction
- **Decision** — branch the flow with if/then conditional logic
- **CRM** — update contact, company, or deal records automatically
- **Communication** — send an email, SMS, WhatsApp message, or make a call
- **Data** — transform, extract, or map data between steps
- **Wait** — pause execution for a set time or until a condition is met
- **Integration** — connect to external tools and services

**Trigger Types**

- **Manual** — run on demand with a single click
- **Webhook** — triggered by an event from an external system
- **Schedule** — run on a cron schedule, daily, weekly, monthly, or after a delay
- **Email** — trigger based on incoming Gmail messages with label or attachment filters

**Workflow Scheduler**

- Set precise cron expressions or choose a simple recurring interval
- Timezone-aware scheduling
- Preview the next scheduled run time before saving

**Execution Monitoring**

- Live status for every run: Running, Completed, Failed, Cancelled, or Timed Out
- Timeline view showing each node's execution in sequence
- Node-level diagnostics to inspect inputs, outputs, and errors
- Filter execution history by success or failure

**Reliability & Control**

- Retry policy: configure max retries with exponential backoff
- Error handling: stop the workflow, continue to the next step, or run a fallback path
- Log retention settings to keep history for as long as you need

**Workflow Management**

- Version history: see and restore previous versions of a workflow
- Clone workflows to reuse as templates
- Import and export workflows across workspaces
- Pre-built workflow templates to get started faster
- Enable or disable workflows with a single toggle

## 🎬 Loom 5 — Agency Portal: White-Label, Branding & Client Management

[Watch on Loom](YOUR_LOOM_LINK_HERE)

### What's Covered

This video explains the Agency Portal — how to white-label the entire SalesAI platform under your own brand, set up a custom domain, and manage multiple clients from a single dashboard. Ideal for agencies that want to resell AI-powered sales tools as their own product.

### Key Features

**Three-Step White-Label Setup**

1. **Sign Up** — create your agency account on SalesAI
2. **Brand It** — upload your logo, set your colors, and configure your domain
3. **Resell** — onboard clients under your brand and start earning recurring revenue

**Branding Panel**

- Upload your agency logo (displayed in the platform header and emails)
- Set a custom favicon for browser tabs
- Upload a separate email logo for all outbound communications
- Choose your brand's primary color — applied across the entire UI
- Set your agency name shown on the login page and throughout the platform

**Custom Domain**

- Add your own domain (e.g., `app.youragency.com`) instead of SalesAI's default URL
- Step-by-step DNS verification instructions with copy-to-clipboard DNS records
- SSL certificate management: request, verify, and track certificate status
- Domain verification status shown in real time (Pending / Verified)

**Multi-Client Dashboard**

- View all client sub-accounts in one place
- See each client's status: Active or Trial
- Track contact count and activity per client
- Monitor combined Monthly Recurring Revenue (MRR) across all accounts

**Agency Billing & Plans**

- Agency-specific plan tiers with higher limits for managing multiple clients
- Billing and payment management from the admin panel
- Feature access gating with clear upgrade prompts

**Team & Permission Controls**

- Manage staff member access per agency domain
- Control which team members can see or manage specific client accounts
- Password and security management for agency-level users

**White-Label Login Page**

- Clients log in through your branded domain with your logo and colors
- No SalesAI branding visible to your end clients
